This is the [Next.js](https://nextjs.org) storefront, deployed from the repo root.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the result.

## Repo layout

- `/` — the Next.js site (this app)
- `scraper/` — the Playwright scraper used to populate `src/data/` (see [scraper/README.md](scraper/README.md))

## Deploy on Vercel

The Next.js app lives at the repo root, so no custom Root Directory setting is needed — deploy this repo directly.
